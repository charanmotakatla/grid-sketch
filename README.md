Grid-Sketch is a simple drawing web application.
It is built using HTML, CSS, and JavaScript.
The project creates a dynamic grid on the screen.
Users can draw by hovering over the grid cells.
The default grid size is 16 x 16.
Users can change the grid size between 8 and 100.
There is a color picker to choose custom colors.
A random color button allows rainbow drawing.
An eraser button helps remove the drawing.
The reset button clears the entire sketch board.
CSS Grid is used to create the layout.
JavaScript is used for DOM manipulation.
The design is responsive for different screen sizes.
This project improves understanding of event listeners.
It is a beginner-friendly front-end practice project.
